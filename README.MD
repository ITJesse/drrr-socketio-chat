Drrr Websocket Chat
=======
Intro
------
Drrr Websocket Chat is a Websocket based online IM platform like drrr-like-chat.

Usage
-----

1.	Use `npm i` to install.
2.	Run with `npm start`
3.	Open `http://your-ip:3000` and become a Dollars!
